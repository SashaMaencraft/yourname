## 📌 Что это?

**VisualVertexRP Mini App** — это удобный интерактивный справочник, встроенный прямо в Telegram.  
Он помогает новичкам и опытным игрокам быстро разобраться в терминологии проекта: какие бывают **зоны** (ЗЗ, ОЗ, КЗ), что означают **статусы** (ЗК, ЧМ), какие **механики** работают на сервере, а также какие **наказания** предусмотрены за нарушения.

Всё собрано в тёмной, комфортной для глаз теме, с поиском и фильтрацией.  
А ещё есть **три главные кнопки** для перехода на сайт, Telegram-канал и партнёрский проект **IFOUS**.

---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 📖 **Словарь терминов** | Все основные термины с пояснениями и RP-примерами |
| 🔍 **Поиск и фильтрация** | Быстрый поиск по словам + фильтр по категориям (Зоны, Статусы, Механика, Валюта) |
| ⚠️ **Выделение наказаний** | Термины, связанные с баном, варном, мутом, подсвечиваются **красным** |
| ⌨️ **Навигация с клавиатуры** | На ПК можно листать страницу стрелками ↑ ↓, PageUp/Down, Home/End |
| 🌐 **Три кнопки быстрого перехода** | Сайт проекта, Telegram-канал, внешний ресурс **IFOUS** |
| ✅ **Верификация проекта** | Интерактивная голубая галочка в футере — при клике показывает, что проект официальный |
| 📱 **Адаптивность** | Идеально работает и на телефоне (Telegram WebApp), и на компьютере |
| 🌙 **Тёмная тема** | Неоновая тёмная гамма, глаза не устают даже при долгом использовании |

## 🖼️ Как это выглядит?

- **Главный экран** — три большие кнопки, поиск, фильтры.
- **Карточки терминов** — название, категория, описание, живой RP-пример.
- **Красные карточки** — термины «БАН», «ВАРН», «ДМ» и другие, связанные с наказаниями.
- **Шпаргалка (таблица)** — краткая версия всех терминов под рукой.
- **Футер с голубой галочкой** — подтверждение верификации проекта.

> Всё стилизовано под единый стиль VisualVertexRP.

---

## 🛠️ Технологии

- **HTML5 / CSS3** — адаптивная вёрстка, тёмная тема, анимации.
- **JavaScript (ES6)** — динамический рендер карточек, поиск, фильтрация, обработка клавиш.
- **Telegram WebApp SDK** — интеграция с Telegram (растягивание на весь экран, haptic-отклик).
- **Netlify / любая статика** — готов к развёртыванию как статический сайт.

## 🚀 Запуск и использование

### Для разработчика:
1. Скачайте файл `index.html`.
2. Откройте его в браузере — всё уже работает локально.
3. Для публикации залейте на любой хостинг (Netlify, GitHub Pages, Vercel).
4. Вставьте ссылку в **BotFather** как WebApp для Telegram-бота.

### Для игроков:
- Просто откройте Mini App внутри Telegram-бота проекта VisualVertexRP.
- Или перейдите по прямой ссылке на хостинг — приложение адаптируется под любое устройство.
